Frequently Asked Questions

What is a spectrum cluster?

A spectrum cluster is a group of Mass Spec spectra that share the same characteristics, namely the shapes of their peaks. The spectra that are similar are grouped together in the PRIDE Cluster pipeline.

Where do the spectrum come from?

The original spectrum comes from PRIDE Archive, which is a submission based data repository.

Due the diversity of PRIDE Archive submissions, we have spectrum from a large number of species and also covers many post translational modifications.

In order to keep the data submitted to us save and secure, we only take the public experiments into account for running the PRIDE Cluster production pipeline.

What can I learn from a cluster?

While grouping spectrum into clusters, we also take into account their original metadata, such as: peptide, modification and species.

For high quality cluster, we try to infer using statistical techniques the following information:

  • Consensus spectrum
  • Consensus peptide
  • Species
  • Modifications

For example, if there is a peptide of your interest, you can use our peptide search to identify whether this peptide has been seen before, in which species it likely to be seen, what are the pontential modifications.

What do the numbers mean for each cluster?

There are three key numbers that can help you to interrpret the cluster:

  • Max ratio: The ratio between the largest number of spectra identify the same peptide and the total number of spectra within a cluster.
  • Number of spectra: The number of spectra within a cluster.
  • Number of projects: The number of original projects from PRIDE Archive.

What is a high quality cluster?

We define high quality clusters using the following criteria:

  • Max ratio >= 0.7: The ratio between the largest number of spectra identify the same peptide and the total number of spectra within a cluster. The ratio must be greater or equal to 0.7.
  • Number of spectra >= 10: The number of spectra within a cluster must be greater or equal to 10.
  • Number of projects >= 2: The number of original projects from PRIDE Archive must be greater or equal to 2.

These criterias are explained in details in the original publication

What is a consensus spectrum?

A derived spectrum from all the spectra within a cluster, this spectrum is theoretically better quality and representative of the identification.

What is a consensus peptide?

If the majority of the spectra within a cluster agrees on a particular peptide identification, we then infer that this peptide is the likely peptide identification given the existing data.

How often is PRIDE Cluster released?

We are planning to run clustering of the new data every three months, the release will follow immediately after we got the clustering results.
