Understand Cluster

For each cluster, we offer a detailed summary page where you can view the key statistics and quliaty metrics of a cluster.

Key cluster statistics

At the top of the cluster details page, you will see the consensus peptide of the cluster and cluster related statistics. The key statistics include:

  • #Spectra: Number of spectra identifing the consensus peptide / Total number of spectra in the cluster.
  • #Projects: Number of projects identifing the consensus peptide / Total number of projects in the cluster.
  • #Species: Number of species assigned to the consensus peptide / Total number of species in the cluster.

Species chart

This chart shows the percentage of species within a cluster. It is important to note that the numbers shown are the number of PSMs. These numbers are likely to be larger than the number of spectra within a cluster.

Modification chart

This chart shows the percentage of modifications within a cluster. It is important to note that the numbers shown are the number of PSMs. These numbers are likely to be larger than the number of spectra within a cluster.

Unique peptides

This table show the number of unique peptide seqences with their modifiations. We also display the following metrics:

  • #PSM: the number of PSMs identify the peptide.
  • #Species: the number of species identify the peptide.
  • #Projects: the number of original projects identify the peptide.

Original projects

The details of the original projects from the PRIDE Archive within the cluster. You can flip thought the projects using the arrows on either side of the panel.

Precursor m/z chart

This boxplot shows the delta m/z distribution for the PSMs identify the consensus peptide and the whole of the cluster.

Delta m/z is the different between the experimental m/z and the theoritical m/z. As a rule of thumb, the smaller of delta m/z the better.

Spectrum similarity chart

This boxplot shows the spectrum similarity distribution between the spectra within the cluster and its consensus spectrum.

For the similarity value, the higher the similarity value the better. A value of 1 means highly similary and almost identical. A value of 0 means not similar at all.

Consensus spectrum

The peak list shows the peaks of the consensus spectrum of the cluster.
