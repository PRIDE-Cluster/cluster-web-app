Search Peptide

How to search?

The search box at the top right corner of the page is the main interface for searching peptide sequences. (see figure below)

Simply type or copy-and-paste the sequence of your interests, then click on the search button.

Interpret the results

The results returned by the search are a summary of a cluster. At the moment, we only show you the high quality clusters. The summary include the following information:

  • Peptide sequence: the consensus peptide sequence for each cluster.
  • Precursor m/z: the consensus precursor m/z for each cluster.
  • Precursor charge: the consensus precursor charge for each cluster.
  • #Spectra: the number of spectra within a cluster.
  • #Projects: the number of original projects within a cluster.
  • #Species: the number of species within a cluster.
  • Max Ratio: The ratio between the largest number of spectra identify the same peptide and the total number of spectra within a cluster.

Filtering

At the moment, we are offering two kinds of filtering to restrict the number of results returned.

  • Filter by species: you can filter the result by the species of your interests.
  • Filter by modifications: you can filter the result by the modifications of your interests.

You can also type into text box above each filter to search for a particular species or modification.
