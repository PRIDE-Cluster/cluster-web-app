Cluster Results Files

These cluster results files are derived from the PRIDE Cluster results. They classified the PRIDE Cluster results in different files that can be use by the community for different analyses and aims. For example the Unidentified clusters with at least 100 spectra can be use to look for new peptide idientifications, variants, modifications that are difficult to identified with common search engine strategies; the Phospho clusters contains all reliable phosphorilated clusters.
