About

PRIDE Cluster contains all public identified experiments in the PRIDE repository and clusters them using an algorithm based on the normalized dot product (see publication). Thereby, similar spectra are grouped together solely based on their peaks.

How to use PRIDE Cluster

You can access PRIDE Cluster in two ways:
  • Search for all clusters that contain spectra identified as a given peptide sequence
  • Download cluster data in the form of Spectrum Libraries

Peptide search

By its peptide search, PRIDE Cluster can help confirm protein identifications.
Just provide the peptide sequence your are investigating and browse the consensual evidence found across high-quality datasets in PRIDE.

Spectrum Libraries

The cluster results can be downloaded in the form of libraries for its use off-line.

PRIDE Cluster Stats

The current Cluster database contains:
  • 3,822,526 clusters
  • 252 species
  • 23,758 experiments
  • 8,662,818 distinct peptides
  • 54,159,998 identified spectra
  • 16,557,497 (30.57%) spectra not in clusters
